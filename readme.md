# Node.js Express Service

A simple Node.js service built with Express.js.

Powered By 

```
npm v9.6.7
node v18.17.1
mysql v8.0
```

## Features

- Express.js server with basic routing
- Middleware for request logging
- JSON response handling
- OpenAPI validator
- Sequelize (MYSQL)
- Supertest test case
- Basic authorization
